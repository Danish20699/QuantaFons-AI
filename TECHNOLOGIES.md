# Technologies Used

This document outlines the key technologies and libraries used in the QuantaFONS-AI project.

## Frontend

- **React**: A JavaScript library for building user interfaces, version 19.2.0.
- **Vite**: A fast build tool and development server for modern web projects.
- **TypeScript**: A typed superset of JavaScript that compiles to plain JavaScript.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Radix UI**: A set of low-level UI primitives for building high-quality design systems.
- **Framer Motion**: A production-ready motion library for React.
- **Three.js**: A JavaScript 3D library used for creating and displaying animated 3D computer graphics.
- **Lucide React**: A beautiful & consistent icon toolkit made by the Lucide community.
- **TanStack Query**: Powerful data synchronization for React.
- **Wouter**: A tiny router for React and Preact.
- **Next Themes**: An abstraction for themes in your React app.
- **Sonner**: A toast notification library for React.

## Backend

- **Express**: A minimal and flexible Node.js web application framework.
- **Node.js**: A JavaScript runtime built on Chrome's V8 JavaScript engine.
- **TypeScript**: Used for server-side development as well.
- **Passport**: Simple, unobtrusive authentication for Node.js.
- **Express Session**: Session middleware for Express.
- **WebSocket (ws)**: A Node.js WebSocket library.

## Database

- **Drizzle ORM**: A TypeScript ORM for SQL databases.
- **Better SQLite3**: A Node.js library for SQLite3 databases.
- **PostgreSQL (pg)**: A PostgreSQL client for Node.js.
- **Connect PG Simple**: A PostgreSQL session store for Express sessions.

## Validation and Forms

- **Zod**: A TypeScript-first schema declaration and validation library.
- **React Hook Form**: Performant, flexible forms with easy validation.
- **Zod Validation Error**: A library to format Zod errors.

## Build and Development Tools

- **Vite**: Build tool and dev server.
- **TypeScript Compiler (tsc)**: TypeScript compiler.
- **ESBuild**: An extremely fast JavaScript bundler.
- **PostCSS**: A tool for transforming CSS with JavaScript.
- **Autoprefixer**: A PostCSS plugin to parse CSS and add vendor prefixes.
- **Tailwind CSS Vite Plugin**: Vite plugin for Tailwind CSS.
- **Drizzle Kit**: A CLI tool for Drizzle ORM.

## UI Components and Utilities

- **Class Variance Authority (CVA)**: A library for creating variant-based component APIs.
- **clsx**: A tiny utility for constructing className strings conditionally.
- **Tailwind Merge**: Utility for merging Tailwind CSS classes.
- **Date-fns**: Modern JavaScript date utility library.
- **Input OTP**: A React component for one-time password input.
- **Embla Carousel**: A lightweight carousel library.
- **Recharts**: A composable charting library built on React components.

## Development and Deployment

- **Replit Plugins**: Various Vite plugins for Replit development environment.
- **Dotenv**: A module that loads environment variables from a .env file.

## Optional Dependencies

- **Bufferutil**: A Node.js module for faster WebSocket parsing.

This list is derived from the project's package.json and configuration files. For the most up-to-date information, refer to the package.json file.
